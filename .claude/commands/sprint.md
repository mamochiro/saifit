# /sprint — Show sprint status dashboard

Read SPRINT.md and produce a concise status report:

1. **Phase summary table** — show all 14 phases with their current status (⬜/🔄/✅/❌)
2. **Current phase** — identify the first phase that is not ✅ and show its remaining unchecked tasks
3. **Progress bar** — X of 14 phases complete
4. **Blockers** — any items marked ❌ with notes
5. **Next 3 tasks** — the next three unchecked `[ ]` items across phases

Keep it scannable. No prose. Use the table + bullets only.
